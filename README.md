# 🏗 Rig

> "Build Go. Keep it clean. Keep it fast."

`rig` is an industrial-strength code generation repository built for developers who reject over-engineering. It uses a single, shared AST parser to extract your core domain models and automatically forge your components.

### Why Rig?
- **Clean Architecture**: Automatically spits out use-cases, repositories, and handlers from your types.
- **One Parser to Rule Them All**: Universal AST processing means blazing-fast generation times.
